## 📋 Agent 工作流程

| 階段 | 動作 |
|------|------|
| 🔴 收到任務 | 新增到「待處理」→ 狀態設 `🔴` |
| 🟡 執行中 | 移到「進行中」→ 狀態改 `🟡` → 逐項打勾驗收 |
| ✅ 完成 | 移到「已完成」→ 狀態改 `✅` → 補完成日期 → 更新 DEV_LOG.md |
| ⚠️ 受阻 | 狀態改 `⚠️` → 補受阻原因 → 通知開發者 |

**任務格式：**
```
### [編號] 標題
- **狀態**：🔴/🟡/✅/⚠️
- **檔案**：`路徑`
- **描述**：說明
- **驗收條件**：
  - [ ] 條件
```
